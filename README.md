# False Lead

False Lead is a multiplayer social deduction party game inspired by games like Impostor Who.

## Features

- Create and join rooms
- Real-time multiplayer with Socket.IO
- Multiple round types
- Voting and scoring system
- Pass-the-phone and online gameplay support

## Tech Stack

- Node.js
- Express
- Socket.IO
- MySQL
- React (in progress)
